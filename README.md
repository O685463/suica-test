# 実写版スイカゲーム (Suica Real)

Vite + Matter.js で作成された、ブラウザで遊べる物理パズルゲームです。11種類の実写フルーツ画像を合成して、スイカを目指しましょう。

## 特徴
- **実写画像**: リアルなフルーツ画像を使用。
- **物理演算**: Matter.js による自然な動き。
- **モバイル対応**: スマートフォンでも快適にプレイ可能。

## 遊び方
1. 画面上部をタップ/クリックして、フルーツを落とす位置を決めます。
2. 指/マウスを離すとフルーツが落ちます。
3. 同じ種類のフルーツがぶつかると、次の大きさのフルーツに進化します。
4. 箱からあふれないように高得点を目指してください。

## 開発
### セットアップ
```bash
npm install
```

### 開発サーバー起動
```bash
npm run dev
```

### ビルド
```bash
npm run build
```

## デプロイ
GitHub Actions により、`main` ブランチへのプッシュ時に自動的に GitHub Pages にデプロイされます。
